## Vs code 주요 단축키
* `ctrl+ k` -> `ctrl +\` 위/아래 수직 화면 분할
* `ctrl + \` 좌우 수평 화면 분할
* `ctrl + J` 터미널 실행/닫기\
* `Q` or `ctrl+c` 터미널 입력 안되는 에러 해결 단축키

## 태그 + 이름 속성 Class, id
* .Class: 반복 유형 분류 시 사용, 반복 지정 가능
* #id: 전체 페이지 중 단하나의 요소에만 지정 시 사용
* class, id는 태그 관계없이 모든 태그에 적용 가능
* 이름 작성 시 주의사항: 반드시 용도에 맞는 의미있는 영단어 사용
